# Hotdog Vendors
Unit 4: Coursework for BTEC Nationals Computing Course (AAQ)

## Style Guide:
* "us_" behind variables/constants/etc. means unsorted
* "s_" behind variables/constants/etc. means sorted
* "v_" behind variables/constants/etc. means vendor
* "_nd" at the end of variables/constants/etc. means name & date

### Variables:
* currenthv: the current hotdogs.txt file data in a variable
